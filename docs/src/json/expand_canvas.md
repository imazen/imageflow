# Expand Canvas Command

The following adds a 10px pink border around the image. 
```json
{
    "expand_canvas": {
        "left": 10,
        "top": 10,
        "right": 10,
        "bottom": 10,
        "color": {
            "srgb": {
              "hex": "FFEECCFF"
            }
        }
    }
}
```
